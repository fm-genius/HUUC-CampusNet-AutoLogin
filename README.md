## ⚠️ 必看前提 (最重要的一步)
脚本能运行的前提，是你的电脑**必须通过物理方式连上了学校网络**！
* **Wi-Fi 用户**：请务必在电脑的 Wi-Fi 列表里，连接学校的校园网信号（如 `HUUC_WLAN` 等），并**勾选“自动连接”**！这样电脑开机时才会先连上 Wi-Fi，脚本才有发挥的空间。
* **网线 用户**：请确保网线已插紧。

## 🛠️ 食用方法一：一键运行版 (强烈推荐，适合所有同学)

不需要懂编程，不需要安装任何环境，纯小白 1 分钟搞定！

### 1. 下载与初始化
1. 在右侧的 [Releases] 页面下载最新版的 `login.exe`。
2. 将它存放在电脑里一个**永远不会移动**的文件夹中（例如 `D:\CampusNet\`）。
3. **双击运行 `login.exe`**。根据黑窗口的提示，输入你的学号和密码并回车。
4. 提示“登录成功”后，程序会自动关闭，并在旁边生成一个 `campus_net_config.txt` 密码本。以后它就完全全自动了！

### 2. 设置后台自动运行 (一劳永逸)
1. 按 `Win + S` 搜索并打开 **任务计划程序**。
2. 点击右侧的 **“创建任务...”**（注意：不要点“创建基本任务”）。
3. **【常规】**：名称填入“校园网自动登录”。
4. **【触发器】**：
   - 点击“新建” -> 选择“登录时” 。
   - 再次“新建” -> 选择“工作站解锁时”（针对睡眠唤醒）。
5. **【操作】**：
   - 点击“新建” -> 操作选择“启动程序”。
   - **程序或脚本**：点击浏览，选中你刚才的 `login.exe`。
   - **起始于(可选)**：⚠️**必填！** 填入 `.exe` 所在的文件夹路径（例如 `D:\CampusNet\`，注意不要带文件名）。
6. **【条件】**（笔记本必看）：**取消勾选**“只有在计算机使用交流电源时才启动此任务”。
7. 点击确定保存！大功告成！

❓ 常见问题 (FAQ)
Q: 为什么我运行程序后，黑窗口闪了一下就没了，网也没连上？
A: 请检查你的 Wi-Fi 是否已经处于连接状态（或正在连接中）。如果没有网络信号，程序会瞬间判定失败并静默退出。

Q: 我改密码了怎么办？
A: 直接把你存放 login.exe 文件夹里的 campus_net_config.txt 文件删除。再次双击 login.exe，它就会重新让你输入新的账号密码。
